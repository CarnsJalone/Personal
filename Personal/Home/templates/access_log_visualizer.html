Apache Access Log Map Visualization

A while back, I saw an interesting idea... Someone examined there NGINX access logs and created a map to visualize the locations of some of those IP Addresses. Well I kind of stole that idea. Here I am using a the Google Maps API to chart the rough locations of those IP addresses. It's really interesting to see how far the reach of the internet is, scam or not. You can also click on each marker to return a small description I gathered about each visitor as well.
